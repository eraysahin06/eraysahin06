## Unity & Game Development

I build **gameplay systems and simulation-driven mechanics in Unity using C#**.

My main interest is creating systems that feel **reactive, systemic, and alive** rather than scripted and static.

### What I build in Unity

- **NPC AI systems** using NavMesh, state-driven behaviors, chase / roam logic, and interaction triggers
- **Mission systems** with objectives, progression, scripted sequences, and world-state changes
- **Dialogue and interaction systems** for events, cutscenes, and player-facing choices
- **Open-world mechanics** including dynamic time-of-day, atmosphere, and environmental feedback
- **Vehicle gameplay** such as driving, biking, and traversal-related systems
- **Gameplay UI** for mission flow, interaction prompts, and in-world feedback
- **Inventory and item systems** tied to progression, objectives, and simulation loops

### Current Unity Focus

- Open-world game structure
- Narrative mission flow
- NPC behavior and pursuit systems
- Vehicle and traversal mechanics
- Interactive world systems
- Scalable gameplay architecture

### Game Design Interests

- immersive open-world gameplay
- narrative-driven systems
- simulation loops
- tension and atmosphere
- emergent gameplay
- systemic interactions
